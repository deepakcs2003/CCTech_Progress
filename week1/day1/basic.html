C++ Programming Notes

1. Introduction to C++

What is C++?

C++ is a general-purpose, high-level programming language that supports procedural, object-oriented, and generic programming paradigms. It was designed to be an extension of the C programming language, adding features like classes, objects, and templates while maintaining compatibility with C code. C++ is compiled, meaning source code is translated into machine code before execution, which makes it efficient and fast. It's widely used for system/software development, game development, and applications requiring high performance.

History of C++

C++ was developed by Bjarne Stroustrup at Bell Labs starting in 1979. It began as "C with Classes" to add object-oriented features to C. The first commercial release was in 1985, named C++. Major standards include:

  • C++98 (ISO/IEC 14882:1998): First standardized version
  • C++03: Minor corrections
  • C++11: Added lambdas, auto keyword, smart pointers, etc.
  • C++14/C++17: Further enhancements like generic lambdas, constexpr
  • C++20: Modules, coroutines, concepts
  • C++23: Recent updates like std::expected, ranges improvements

Features of C++

  • Object-Oriented Programming (OOP): Supports classes, inheritance, polymorphism, encapsulation
  • Procedural Programming: Functions, loops, conditionals
  • Generic Programming: Templates for type-independent code
  • Low-Level Access: Pointers, memory management
  • Standard Template Library (STL): Containers (vector, map), algorithms (sort), iterators
  • Multi-Paradigm: Mix styles as needed
  • Portability: Compiles to native code for various platforms
  • Performance: Close to hardware, no garbage collection (manual memory via new/delete)
  • Exception Handling: try-catch blocks
  • Lambda Expressions: Anonymous functions (from C++11)

Applications of C++

  • System software (OS kernels like Windows parts)
  • Game development (Unreal Engine)
  • Embedded systems (IoT, automotive)
  • High-performance computing (simulations, AI)
  • Web browsers (Chrome's V8 engine)
  • Databases (MySQL)
  • Graphics (OpenGL applications)
  • Financial systems (trading platforms)

Difference between C and C++

  • Paradigm: C is procedural; C++ adds OOP and generic
  • Data Hiding: C uses structs; C++ has classes with access specifiers (public/private/protected)
  • Functions: C++ supports overloading; C does not
  • References: C++ has references (&); C uses pointers only
  • Namespace: C++ has namespaces to avoid name conflicts; C does not
  • I/O: C uses printf/scanf; C++ uses cin/cout (stream-based)
  • Memory: Both manual, but C++ has new/delete vs C's malloc/free
  • Exceptions: C++ has built-in; C uses error codes
  • Compatibility: C++ can compile most C code, but not vice versa

Compilation Process

C++ compilation involves:

  1. Preprocessing: Handles #include, #define, macros. Outputs expanded source (.i file)
  2. Compilation: Translates to assembly code (.s file). Checks syntax, types
  3. Assembly: Converts assembly to object code (.o/.obj file)
  4. Linking: Combines object files, libraries into executable (.exe)

C++ Program Structure

A basic C++ program includes:

  • Header includes (#include <iostream>)
  • Namespace (using namespace std;)
  • Main function (int main() {})
  • Statements ending with ;
  • Braces {} for blocks

Code example:

#include <iostream>  // Header file
using namespace std; // Namespace

int main() {         // Main function
    cout << "Hello, World!";  // Statement
    return 0;        // Return value
}

2. Environment Setup

Installing Compiler (GCC / MinGW)

  • GCC (GNU Compiler Collection): Free, open-source. Install via package manager (apt on Linux, brew on Mac)
  • MinGW (Minimalist GNU for Windows): GCC port for Windows. Download from mingw.org, add to PATH

IDEs (CodeBlocks, VS Code, Visual Studio)

  • CodeBlocks: Free, cross-platform, built-in compiler
  • VS Code: Lightweight, extensions for C++ (Microsoft C++ extension), debugger
  • Visual Studio: Full-featured, Windows-focused, with IntelliSense, debugging

Writing First C++ Program

Create hello.cpp:

#include <iostream>
using namespace std;

int main() {
    cout << "Hello, C++!" << endl;
    return 0;
}

Compile vs Run

  • Compile: g++ hello.cpp -o hello (creates executable)
  • Run: ./hello (executes)

Errors (Syntax, Runtime, Logical)

  • Syntax: Compilation fails (e.g., missing ;). Fix: Check code
  • Runtime: Runs but crashes (e.g., divide by zero). Fix: Use try-catch or checks
  • Logical: Wrong output (e.g., incorrect loop). Fix: Debugging with prints or tools like gdb

3. Basic Syntax

Header Files

Include libraries: #include <header> (system) or "header" (local).

main() Function

Entry point: int main(int argc, char* argv[]) { ... }

using namespace std

Avoids prefixing std:: for standard items like cout.

Comments (Single-Line, Multi-Line)

  • Single: // Comment
  • Multi: /* Comment */

Keywords

Reserved words: int, class, if, while, etc. Can't use as identifiers.

Identifiers

Names for variables/functions: Start with letter/_, alphanumeric.

4. Data Types & Variables

Built-in Data Types

  • int (4 bytes, -2^31 to 2^31-1)
  • float (4 bytes, single precision)
  • double (8 bytes, double precision)
  • char (1 byte, 'a')
  • bool (1 byte, true/false)
  • void (no value)

Code:

int x = 5;
float y = 3.14f;

Derived Data Types

  • Array: Fixed size, e.g., int arr[5];
  • Pointer: Stores address, int* p = &x;
  • Reference: Alias, int& ref = x;
  • Function: Return type, params

Code:

int arr[3] = {1,2,3};
int* ptr = arr;

User-Defined Data Types

  • struct: Group variables, struct Point {int x,y;};
  • class: Like struct with methods
  • union: Shares memory
  • enum: Named integers, enum Color {RED=1};

Code:

struct Person {
    string name;
    int age;
};

Type Modifiers

signed/unsigned (for int/char), short/long (size), const (immutable).

Code:

unsigned int u = 10;
const int c = 5;

Constants (const)

Immutable: const int MAX = 100;

Type Casting

Convert types: static_cast<int>(3.14), reinterpret_cast, const_cast, dynamic_cast (for polymorphism).

Code:

double d = 5.5;
int i = static_cast<int>(d);  // i=5

Size of Data Types

Use sizeof(type) or sizeof(var).

Code:

cout << sizeof(int);  // Typically 4

5. Input / Output

cin

Input: cin >> var;

Code:

int num;
cin >> num;

cout

Output: cout << "Text" << var;

Code:

cout << "Hello" << endl;

getline

Read line: getline(cin, str);

Code:

string line;
getline(cin, line);

Escape Sequences

\n (newline), \t (tab), \" (quote).

Code:

cout << "Hello\nWorld";

Formatting Output

Use <iomanip>: setw(width), setprecision(digits), fixed.

Code:

#include <iomanip>
cout << setw(10) << setprecision(2) << fixed << 3.14159;

6. Operators

Arithmetic Operators

+ - * / % (modulo)

Code:

int a=10, b=3;
cout << a + b << " " << a % b;  // 13 1

Relational Operators

== != > < >= <=

Code:

bool res = (a > b);  // true

Logical Operators

&& (and), || (or), ! (not)

Code:

if (a > 0 && b > 0) {}

Assignment Operators

= += -= *= /= %=

Code:

a += 5;  // a=15

Increment / Decrement

++ -- (pre/post)

Code:

int x=5;
cout << ++x << " " << x++;  // 6 6 (then x=7)

Bitwise Operators

& | ^ ~ << >>

Code:

int res = 5 & 3;  // 1 (101 & 011 = 001)

Ternary Operator

condition ? true : false

Code:

int max = (a > b) ? a : b;

Operator Precedence

() highest, then * / %, + -, << >>, relational, logical, = lowest.

7. Control Statements

Conditional Statements

if

If true, execute.

Code:

if (a > 0) {
    cout << "Positive";
}

if-else

Else if false.

Code:

if (a > 0) cout << "Positive";
else cout << "Non-positive";

else-if

Chain conditions.

Code:

if (a > 0) cout << "Positive";
else if (a < 0) cout << "Negative";
else cout << "Zero";

Nested if

If inside if.

Code:

if (a > 0) {
    if (a % 2 == 0) cout << "Even positive";
}

switch-case

Multi-way branch.

Code:

switch (a) {
    case 1: cout << "One"; break;
    case 2: cout << "Two"; break;
    default: cout << "Other";
}

Looping Statements

for

Init; condition; increment.

Code:

for (int i=0; i<5; i++) {
    cout << i << " ";
}

while

Condition checked before.

Code:

int i=0;
while (i<5) {
    cout << i++ << " ";
}

do-while

Executes once, then checks.

Code:

int i=0;
do {
    cout << i++ << " ";
} while (i<5);

Nested Loops

Loop inside loop.

Code:

for (int i=0; i<3; i++) {
    for (int j=0; j<3; j++) {
        cout << i << j << " ";
    }
    cout << endl;
}

break

Exit loop/switch.

Code:

for (int i=0; i<10; i++) {
    if (i==5) break;
    cout << i;
}

continue

Skip to next iteration.

Code:

for (int i=0; i<5; i++) {
    if (i%2==0) continue;
    cout << i;  // 1 3
}
