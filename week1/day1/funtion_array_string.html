C++ Programming Notes

8. Functions

Functions in C++ are reusable blocks of code that perform specific tasks. They promote modularity, code reuse, and readability. A function can take inputs (parameters), process them, and return a result. Functions can be user-defined or from libraries.

Function Declaration

A declaration (prototype) tells the compiler about the function's name, return type, and parameters without the body. It's placed before use, often in headers.

Syntax: return_type function_name(parameter_list);
int add(int a, int b);  // Declaration

Function Definition

The definition includes the body with actual code.

Syntax: return_type function_name(parameter_list) { // Body return value; // If not void }
int add(int a, int b) {  // Definition
    return a + b;
}

Function Call

Invoking the function with arguments.

Syntax: function_name(arguments);
int main() {
    int result = add(5, 3);  // Call
    cout << result;  // Outputs 8
    return 0;
}

Return Types

Specifies what the function returns: int, double, void (no return), etc. Use return statement.

double divide(int x, int y) {
    return static_cast<double>(x) / y;  // Returns double
}

Parameters

Inputs to functions. Can be by value, reference, pointer, or const.

void print(int num) {  // Parameter 'num'
    cout << num;
}

Call by Value

Copies argument value to parameter. Changes inside function don't affect original.

void increment(int x) {
    x++;  // Only local x changes
}

int main() {
    int a = 5;
    increment(a);
    cout << a;  // Still 5
}

Call by Reference

Passes reference (&), so changes affect original. Efficient for large data.

void increment(int& x) {
    x++;  // Original changes
}

int main() {
    int a = 5;
    increment(a);
    cout << a;  // 6
}

Inline Functions

Compiler replaces call with body for efficiency (no call overhead). Use inline keyword.

inline int max(int a, int b) {
    return (a > b) ? a : b;
}

Default Arguments

Parameters with default values if not provided in call. Must be rightmost.

int add(int a, int b = 0) {  // b defaults to 0
    return a + b;
}

int main() {
    cout << add(5);  // 5 (b=0)
    cout << add(5, 3);  // 8
}

Function Overloading

Multiple functions with same name but different parameters (number/types/order).

int add(int a, int b) { return a + b; }
double add(double a, double b) { return a + b; }

int main() {
    cout << add(5, 3);  // int version
    cout << add(5.5, 3.3);  // double
}

Recursion

Function calls itself. Needs base case to stop.

int factorial(int n) {
    if (n <= 1) return 1;  // Base case
    return n * factorial(n - 1);  // Recursive call
}

int main() {
    cout << factorial(5);  // 120
}

9. Arrays

Arrays are fixed-size collections of same-type elements, stored contiguously in memory. Accessed via index (0-based).

One-Dimensional Array

Linear array.

Declaration: type array_name[size];
int arr[5];  // Declares array of 5 ints
arr[0] = 10;  // Assignment
cout << arr[0];  // Access

Two-Dimensional Array

Matrix-like, rows and columns.

Declaration: type array_name[rows][cols];
int matrix[3][4];  // 3 rows, 4 cols
matrix[1][2] = 5;  // Row 1, col 2

Multi-Dimensional Array

More than 2D, e.g., 3D tensor.

Declaration: type array_name[d1][d2][d3];
int cube[2][3][4];  // 2 layers, 3 rows, 4 cols
cube[0][1][2] = 10;

Array Initialization

Assign values at declaration.

int arr[5] = {1, 2, 3, 4, 5};  // Full init
int partial[5] = {1, 2};  // Rest 0
int zero[5] = {};  // All 0

Passing Array to Function

Arrays decay to pointers. Pass size separately.

void printArray(int arr[], int size) {  // Or int* arr
    for (int i = 0; i < size; i++) {
        cout << arr[i] << " ";
    }
}

int main() {
    int myArr[3] = {1, 2, 3};
    printArray(myArr, 3);
}

Array vs Pointer

Arrays are fixed; pointers variable. Array name is pointer to first element.

int arr[3] = {1,2,3};
int* ptr = arr;  // ptr points to arr[0]
cout << *(ptr + 1);  // 2 (arr[1])

10. Strings

Strings are sequences of characters. C++ supports C-style (null-terminated char arrays) and std::string class (safer, dynamic).

C-Style Strings

Char array ending with '\0'.

char str[6] = "Hello";  // Includes '\0'
cout << str;  // Hello

String Class

Dynamic, resizable from <string>.

#include <string>
string s = "Hello";
s += " World";  // Concat
cout << s;  // Hello World

String Input/Output

cin/cout for string class; getline for lines.

string input;
cin >> input;  // Word (stops at space)
getline(cin, input);  // Full line
cout << input;

String Functions

Methods like length(), empty(), at().

string s = "Hello";
cout << s.length();  // 5
cout << s.at(0);  // 'H'
cout << s.substr(1, 3);  // "ell"

String Comparison

==, !=, <, >, compare().

string s1 = "Apple", s2 = "Banana";
if (s1 < s2) cout << "Apple before Banana";  // Lexicographical
int res = s1.compare(s2);  // Negative if s1 < s2

String Manipulation

Concat, insert, erase, etc.

string s = "Hello";
s.append(" World");  // Hello World
s.insert(5, ",");  // Hello, World
s.erase(5, 1);  // Hello World
s.replace(6, 5, "Universe");  // Hello Universe
