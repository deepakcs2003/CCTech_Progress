1Ô∏è‚É£ unordered_map Key-Value Hash Table

‚úÖ Theory

  • Stores key-value pairs
  • Keys are unique (no duplicate keys)
  • No sorted order
  • Uses hash table
  • Supports [] operator for direct access

‚è±Ô∏è Average Time Complexity

Operation Complexity
Insert O(1)
Access ([]) O(1)
Find O(1)
Erase O(1)
Traverse O(n)

üìå Use when:

  • You need key-value storage
  • Fast lookup by key is required
  • Order does not matter
  • Keys are unique

‚úÖ Syntax

#include <unordered_map>

unordered_map<Key, Value> name;

üîé Syntax Meaning

unordered_map ‚Üí container type

Key ‚Üí data type of key (int, string, etc.)

Value ‚Üí data type of value (int, string, etc.)

name ‚Üí object name

‚úÖ All Important Member Functions

#include <bits/stdc++.h>
using namespace std;

int main() {

    unordered_map<int, string> um;

    // insert({key, value})
    um.insert({1, "Apple"});
    // Meaning: Insert key-value pair

    // [] operator - direct access/insert
    um[2] = "Banana";
    // Meaning: Insert or update value for key 2
    // If key exists ‚Üí update value
    // If key doesn't exist ‚Üí insert new pair

    // Accessing value
    cout << um[1] << endl;
    // Meaning: Get value associated with key 1
    // Returns "Apple"

    // size()
    cout << um.size() << endl;
    // Meaning: Returns number of key-value pairs

    // empty()
    cout << um.empty() << endl;
    // Meaning: Returns true if map is empty

    // find(key)
    auto it = um.find(1);
    // Meaning: Returns iterator to key-value pair if found,
    // otherwise returns um.end()

    if(it != um.end()) {
        cout << it->first << " : " << it->second;
        // first ‚Üí key, second ‚Üí value
    }

    // count(key)
    cout << um.count(1) << endl;
    // Meaning: Returns 1 if key exists, else 0

    // at(key)
    cout << um.at(1) << endl;
    // Meaning: Access value with bounds checking
    // Throws exception if key doesn't exist

    // erase(key)
    um.erase(1);
    // Meaning: Remove key-value pair by key

    // erase(iterator)
    auto itr = um.find(2);
    if(itr != um.end())
        um.erase(itr);
    // Meaning: Remove element at iterator position

    // clear()
    um.clear();
    // Meaning: Removes all key-value pairs

    // Traversing unordered_map
    for(auto pair : um) {
        cout << pair.first << " -> " << pair.second << endl;
    }

    return 0;
}

‚ö†Ô∏è Important: [] vs at()

um[key] ‚Üí Creates key if it doesn't exist (with default value)

um.at(key) ‚Üí Throws exception if key doesn't exist (safer)

2Ô∏è‚É£ unordered_set Hash Table

‚úÖ Theory

  • Stores unique elements
  • No duplicate allowed
  • No sorted order
  • Uses hash table

‚è±Ô∏è Average Time Complexity

Operation Complexity
Insert O(1)
Find O(1)
Erase O(1)
Traverse O(n)

üìå Use when:

  • You need fast lookup
  • Order does not matter

‚úÖ Syntax

#include <unordered_set>

unordered_set<T> name;

üîé Syntax Meaning

unordered_set ‚Üí container type

T ‚Üí data type (int, string, etc.)

name ‚Üí object name

‚úÖ All Important Member Functions

#include <bits/stdc++.h>
using namespace std;

int main() {

    unordered_set<int> us;

    // insert(value)
    us.insert(10);  
    // Meaning: Insert element into set (if duplicate ‚Üí ignored)

    us.insert(20);
    us.insert(10);  
    // Duplicate ‚Üí will NOT insert

    // size()
    cout << us.size() << endl;
    // Meaning: Returns number of elements

    // empty()
    cout << us.empty() << endl;
    // Meaning: Returns true if container is empty

    // find(value)
    auto it = us.find(20);
    // Meaning: Returns iterator to element if found,
    // otherwise returns us.end()

    if(it != us.end())
        cout << "Found\n";

    // count(value)
    cout << us.count(10) << endl;
    // Meaning: Returns 1 if element exists, else 0

    // erase(value)
    us.erase(10);
    // Meaning: Remove element by value

    // erase(iterator)
    us.erase(us.begin());
    // Meaning: Remove element by position

    // clear()
    us.clear();
    // Meaning: Removes all elements

    // bucket_count()
    cout << us.bucket_count();
    // Meaning: Total number of hash buckets

    return 0;
}

3Ô∏è‚É£ unordered_multiset Allows Duplicates

‚úÖ Theory

  • Same as unordered_set
  • But allows duplicates
  • Uses hashing
  • No sorted order

unordered_set

Unique only

unordered_multiset

Duplicate allowed

‚úÖ Syntax

#include <unordered_set>

unordered_multiset<T> name;

‚úÖ All Important Member Functions

#include <bits/stdc++.h>
using namespace std;

int main() {

    unordered_multiset<int> ums;

    // insert()
    ums.insert(5);
    ums.insert(5);
    // Meaning: Duplicate allowed

    // size()
    cout << ums.size() << endl;
    // Meaning: Total elements including duplicates

    // count(value)
    cout << ums.count(5);
    // Meaning: Returns number of occurrences of 5

    // find(value)
    auto it = ums.find(5);
    // Meaning: Returns iterator to first occurrence

    // erase(value)
    ums.erase(5);
    // Meaning: Deletes ALL occurrences of 5

    // erase(iterator)
    auto itr = ums.find(5);
    if(itr != ums.end())
        ums.erase(itr);
    // Meaning: Deletes only ONE occurrence

    // clear()
    ums.clear();
    // Meaning: Remove all elements

    return 0;
}

4Ô∏è‚É£ unordered_multimap Key-Value Pairs

‚úÖ Theory

  • Stores key-value pairs
  • Allows duplicate keys
  • Uses hashing
  • No order maintained
  • Does NOT support [] operator

üìå Why no [] operator?

Because multiple values may exist for same key.

‚úÖ Syntax

#include <unordered_map>

unordered_multimap<Key, Value> name;

‚úÖ All Important Member Functions

#include <bits/stdc++.h>
using namespace std;

int main() {

    unordered_multimap<int, string> umm;

    // insert({key,value})
    umm.insert({1, "A"});
    umm.insert({1, "B"});
    // Meaning: Same key allowed

    // size()
    cout << umm.size() << endl;
    // Meaning: Total key-value pairs

    // find(key)
    auto it = umm.find(1);
    // Meaning: Returns iterator to FIRST element with key=1

    if(it != umm.end())
        cout << it->second;

    // count(key)
    cout << umm.count(1);
    // Meaning: Number of elements with key=1

    // equal_range(key)
    auto range = umm.equal_range(1);
    // Meaning: Returns pair of iterators
    // covering ALL elements with that key

    for(auto itr = range.first; itr != range.second; itr++)
        cout << itr->second << " ";

    // erase(key)
    umm.erase(1);
    // Meaning: Delete ALL elements with key=1

    // erase(iterator)
    auto itr = umm.begin();
    umm.erase(itr);
    // Meaning: Delete only that element

    // clear()
    umm.clear();
    // Meaning: Remove all key-value pairs

    return 0;
}
